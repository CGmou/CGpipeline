# CGPipeline: The Artist Dashboard

CGPipeline is a task-centric VFX/CG pipeline manager designed for speed and simplicity and cross multi Platform like Windows and MacOS.

## Core Philosophy
1. **Flat UI:** No folder trees. Tasks are presented as cards.
2. **One-Click Launch:** Artists click "CONTINUE WORK" to instantly go to their task files.
3. **Visual Progress:** Every card supports thumbnails and status tracking.
4. **Integrate with Kitsu!!!**

For now only works with Blender in Windows and MacOS.
For Maya 2027 only works in Windows env.


Build under Blender 5.1.1
Build under Maya 2027 (Windows)
Build with Kitsu
